Цель проекта: создать веб-сайт с интерактивной картой кладбища. 
Функционал:
1) Полностью на русском языке
2) Интерактивная карта, на которой можно отмечать места захоронений
3) Возможность передавать координаты напрямую с телефона человека, для определения местоположения захоронения
4) Авторизация пользователей
5) Заметки для каждого пользователя, чтобы он мог отмечать интересные ему места (например местонахождение определенных захоронений) 
6) На карте отображается множество точек - захоронений. При нажатии на точку должна появляться информация о захоронении
6.1) Даты рождения и смерти человека
6.2) ФИО человека
6.3) Возможность добавить описание (уникальное для каждого пользователя)
6.4) Должна быть возможность для каждого пользователя добавить определенные захоронения в отдельный список для дальнейшего быстрого нахождения
6.4.1) Выбранные пользователем захоронения должны быть выделены на карте
7) доложен быть поиск
7.1) По ФИО
7.2) По датам рождения/смерти
7.3) По выбранным ранее захоронениям
8) Две роли: Администратор и Пользователь. 
8.1) Администратор может добавлять описание к захоронениям, которые будут видны всем пользователям
8.2) Администратор может добавлять новые захоронения
8.3) Пользователи отправляют координаты с мобильного телефона на модерацию, после проверки подлинности администратор может добавить захоронение, которое прислал пользователь или отменить
8.3.1) в случае отказа в личный кабинет пользователя должно прийти соответствующие письмо с информацией о том, что его запрос на добавление нового захоронения был отклонён
8.3.2) в случае одобрения, пользователю приходит письмо с информацией о том, что предложенное им захоронение было добавлено на карту
9) API ключ Яндекс карт, которые используются в качестве карт в проекте: 69436f8e-f6b6-41bf-9606-13f8a73cc804
10) В качестве первого кладбища было выбрано Читинское центральное кладбище с координатами: 52.033635, 113.501049

Технологии, которые должны использоваться: python, Django, rest API, SQLite3, HTML, CSS, JS и другие при необходимости.
